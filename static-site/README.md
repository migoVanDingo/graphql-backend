# Shao-Lin New Mexico - Static React Site

This is a modern React conversion of the legacy Shaolin New Mexico PHP website.

## Pages

- **Home** (`/`) - Landing page with welcome message, testimonials carousel
- **Classes** (`/classes`) - Information about curriculum, instruction, wooden man training, and facilities
- **Membership** (`/membership`) - Membership tiers, pricing, and contact form
- **Schedule** (`/schedule`) - Weekly class schedule and 4-month advanced material schedule
- **About** (`/about`) - Information about instructors, founders, and lineage

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- React Router DOM
- CSS Modules
